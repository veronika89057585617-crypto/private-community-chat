Запуск на MacBook

Mac блокирует двойной клик по файлам .command. Запускайте через Terminal.

  1. Откройте Terminal через Command + Пробел.
  2. Скопируйте команду ниже.
  3. Вставьте в Terminal и нажмите Enter.
cd "$HOME/Downloads/ANON_CHAT_MAC_SAFE_START" && bash START_MAC.command

PIN-кода нет. Админка откроется по секретной ссылке автоматически.
