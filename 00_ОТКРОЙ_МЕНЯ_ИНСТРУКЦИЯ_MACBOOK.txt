ИНСТРУКЦИЯ ДЛЯ MACBOOK — ЛОКАЛЬНЫЙ ТЕСТ

ВАЖНО:
Если сайт запущен только на MacBook, он доступен как http://localhost:...
Такая ссылка работает на этом MacBook, но НЕ будет нормально открываться у участников с телефона через Telegram.
Чтобы участник нажал ссылку в Telegram и сразу попал в чат, сайт нужно выложить на публичный сервер/хостинг.

ЛОКАЛЬНО ЗАПУСТИТЬ ДЛЯ ТЕСТА:
1. Откройте Terminal: Command + Пробел → Terminal → Enter.
2. Скопируйте и вставьте команду:

cd "$HOME/Downloads/ANON_CHAT_MAC_SAFE_START_v3" && bash START_MAC.command

3. Нажмите Enter.
4. Браузер откроет админскую ссылку.
5. Terminal не закрывайте, пока пользуетесь сайтом.

ЧТО НОВОГО:
- PIN-кода нет.
- Админ входит по секретной ссылке.
- Одна ссылка участника закрепляется только за одним устройством.
- Админ видит список онлайн: имя/метка участника + ID.
- Участники видят только количество онлайн.
- Можно удалять чаты.
- Можно менять название, описание, категорию, картинку и запрещённые слова чата.
- Нарушения видны администратору в отдельном блоке.
- ФИО, компании, адреса, телефоны, email, ссылки и ники блокируются.

ДЛЯ РЕАЛЬНЫХ УЧАСТНИКОВ:
Нужно разместить проект на сервере. Тогда ссылки будут вида:
https://ваш-сайт.com/join/...

Только такая публичная ссылка откроется у участника с телефона в Telegram.
