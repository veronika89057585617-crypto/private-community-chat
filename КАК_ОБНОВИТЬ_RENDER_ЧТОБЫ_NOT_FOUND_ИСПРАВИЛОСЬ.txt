ЧТО ИСПРАВЛЕНО

Ошибка Not Found появлялась потому, что сервер не отдавал страницу сайта по адресам:
/admin/...
/join/...

В этой версии исправлено:
- админские ссылки /admin/... открываются;
- ссылки участников /join/... открываются;
- в Render logs будет показываться правильная публичная Admin link, если Render передаёт RENDER_EXTERNAL_URL.

КАК ОБНОВИТЬ САЙТ НА RENDER

1. Откройте GitHub.
2. Откройте репозиторий private-community-chat.
3. Загрузите туда файлы из этой папки с заменой старых.
4. Нажмите Commit changes.
5. Render автоматически начнёт новый deploy.
6. Дождитесь статуса Live.
7. Откройте новую Admin link из Logs.

Если Render не начал deploy сам:
- зайдите в Render;
- откройте private-community-chat;
- нажмите Manual Deploy;
- выберите Deploy latest commit.
